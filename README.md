# text2smpl

generate 3D body model on SMPL template from text description

**Requirements:**

tensorflow-gpu 1.13.1 \
cudatoolkit 9.0 \
opendr \
scipy \
numpy \
Cython \
easydict \
future 

**run the test example:**
cd text2smpl-master \ 
python main.py 

And you'll find the output 3D SMPL bodies in foler "*/data/objs"
